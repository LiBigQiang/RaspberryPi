# git server

搭建git服务器

## 参考文档

* [Setting up Your Raspberry Pi as a Git Server](https://www.sitepoint.com/setting-up-your-raspberry-pi-as-a-git-server/)

## steps

* `sudo adduser git`
* `su git`
* `mkdir helloworld.git`
* `cd helloworld.git`
* `git init --bare`
* `git clone git@192.168.31.189:/home/git/helloworld.git`
