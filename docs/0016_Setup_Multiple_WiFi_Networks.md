# Setup Multiple WiFi Networks

设置Wifi账户密码

* [RPi 3B 无线连接配置](https://www.cnblogs.com/zengjfgit/p/8854449.html)

## 操作方法

* `/etc/wpa_supplicant/wpa_supplicant.conf`
  ```
  ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
  update_config=1
  country=GB
  
  network={
          ssid="your ssid"
          psk="your password"
          key_mgmt=WPA-PSK
  }
  
  network={
          ssid="your ssid"
          psk="your password"
          key_mgmt=WPA-PSK
  }
  ```
