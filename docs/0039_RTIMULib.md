# RTIMULib

尝试一下RTIMULib库的使用方法

## Source code

https://github.com/RPi-Distro/RTIMULib/tree/master/Linux

## Dependens

* `sudo apt-get install cmake`
* `sudo apt-get install libqt4-dev`
